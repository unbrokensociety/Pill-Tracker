# 🚀 PillTracker v1.23 — Release Notes

### 🌟 What's New in Version 1.23

* ⚡ **Automatic Cloud & Local Database Synchronization**: All medication additions, deletions, refills, and intake log updates are now instantly and automatically synchronized with Firebase Cloud Firestore and local JSON backup cache!
* 🔐 **Strict Secure Authentication & Firebase Password Validation**: Restructured the sign-in security layer. Authentication now rigorously validates credentials against Firebase Auth before granting access, completely preventing bypasses.
* 🔑 **Password Reset ("Forgot Password") Flow**: Added a direct "Forgot Password?" recovery button on the login screen, allowing users to receive password reset links directly to their inbox.
* 📊 **Smart Clinical Medical Report**: Enhanced doctor report exports with adherence compliance percentages, side effect logs, inventory stock alerts, and physician signature templates.
* 🩺 **Dual-Format Doctor Barcode & QR Code**: Doctor scanner card now supports toggling between 2D QR Code and 1D Barcode (Code 128) with integrated real-time patient prescription summaries.
* 📜 **Multi-Language Legal & Privacy Policy**: Added full Privacy Policy and Terms of Service sections in English, Ukrainian, and Russian directly to the repository README.
