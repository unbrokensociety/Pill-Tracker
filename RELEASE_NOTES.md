# 🚀 PillTracker v1.22

### 🌟 What's New in v1.22

* 🔐 **Seamless Email Verification**: When registering via Email & Password, a verification email is dispatched automatically with immediate feedback.
* 🚀 **Instant Access (Offline Mode)**: Added a **"Continue (Offline Mode)"** option so you can start tracking medications right away without waiting for email link confirmation.
* 📩 **Spam Folder Guidance**: Clear UI notes reminding users to check their SPAM folder if the confirmation email is delayed.
* 🛡 **Enhanced Password Security**: Updated minimum password requirements to 6+ characters for better account protection.
* ⚙️ **CI/CD & Test Stability**: Fully updated unit and screenshot tests to ensure reliable, high-quality automated GitHub Actions release builds.
* 🌍 **Localization Sync**: Updated and aligned language resources across all supported locales.
