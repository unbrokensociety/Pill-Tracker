## 🇷🇺 Русский

### 🌟 Что нового в PillTracker v1.22:

* 🔐 **Улучшенный вход и регистрация**: При создании аккаунта по Email/Паролю письмо со ссылкой подтверждения отправляется мгновенно в автоматическом режиме.
* 🚀 **Мгновенный вход (Локальный режим)**: Добавлена кнопка **«Продолжить в приложение (Локальный режим)»** — теперь вы можете использовать все функции PillTracker сразу, не дожидаясь клика по ссылке в письме!
* 📩 **Забота о письмах**: В интерфейс добавлены наглядные подсказки с напоминанием проверить папку **«СПАМ»**, если письмо задерживается почтовым сервером.
* 🛡 **Безопасность**: Валидация пароля обновлена до стандартного уровня безопасности (минимум 6 символов).
* ⚙️ **Идеальный CI/CD**: Исправлены и проверены все автосборки и юнит-тесты на GitHub Actions — APK собирается со 100% успехом!
* 🌍 **Локализация**: Полная синхронизация переводов (Украинский, Русский, Английский).

## 🇺🇦 Українська

### 🌟 Що нового у PillTracker v1.22:

* 🔐 **Оновлена реєстрація**: Під час створення акаунта за Email/Паролем лист із посиланням підтвердження надсилається миттєво та автоматично.
* 🚀 **Швидкий доступ (Локальний режим)**: Додано кнопку **«Продовжити в застосунок (Локальний режим)»** — можна одразу користуватися PillTracker, не чекаючи підтвердження пошти!
* 📩 **Увага до листів**: Додано зрозумілі підказки з проханням перевірити папку **«СПАМ»**, якщо лист затримується.
* 🛡 **Безпека**: Оновлено мінімальну довжину пароля до 6 символів за стандартами безпеки.
* ⚙️ **Надійність збірки**: Оптимізовано автозбірки GitHub Actions та юнит-тести Robolectric — APK будується ідеально.
* 🌍 **Багатомовність**: Повністю актуалізовано переклади (Українська, Англійська, Російська).

## 🇬🇧 English

### 🌟 What's New in PillTracker v1.22:

* 🔐 **Seamless Verification**: Creating an account with Email & Password automatically triggers the verification email instantly.
* 🚀 **Instant Offline Mode**: Added a convenient **"Continue (Offline Mode)"** option to use PillTracker immediately without waiting for link verification.
* 📩 **Spam Inbox Guidance**: Added visual reminders encouraging users to check their **SPAM** folder if the email is delayed.
* 🛡 **Enhanced Security**: Updated minimum password requirements to 6+ characters.
* ⚙️ **Flawless CI/CD Pipeline**: Fixed and verified all unit tests and GitHub Actions workflows for reliable automated APK releases.
* 🌍 **Multilingual Support**: Fully synchronized string resources for English, Ukrainian, and Russian locales.
